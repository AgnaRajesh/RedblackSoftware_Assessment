# RedblackSoftware_Assessment
Assessment for Red black Software
Created using Playwright with TypeScript
# SETUP
1. Install VS Code
2. Create a folder inside system and open it in VS
3. Setup playwright --> npm init playwright@latest
4. Create a folder inside framework to store page files (.ts files)
5. Inside tests folder, create a file with extension ".spec.ts"

# Run command
npx playwright test tests/"Name of the spec file"
